# 🧵 Texora Production Analytics Report

## 📊 Project Overview
This project presents an end-to-end **Production Analytics Report** developed for a textile manufacturing use case (**Texora**). It focuses on transforming raw production data into structured insights to monitor **output, quality, and operational efficiency**.

---

## 🎯 Business Objective
- Limited visibility into **production performance and quality metrics**
- Inefficient tracking of **unit-wise and line-wise output**
- Lack of structured insights for **quality and grade distribution**

---

## 🛠️ Solution Approach

### 🔹 Data Processing
- Performed data cleaning and transformation using **Power Query**
- Standardized datasets to ensure consistency and reliability

### 🔹 Data Modeling
- Developed an optimized data model for efficient analysis
- Created key performance metrics using **DAX**

### 🔹 Report Development
- Designed an interactive **Power BI report** with dynamic filtering
- Enabled analysis across **units, time periods, and production lines**

---

## 📈 Key Analysis Areas

- **Production Performance** – Yearly, monthly, and daily output tracking (MT)  
- **Unit Comparison** – Performance analysis across manufacturing units  
- **Line Efficiency** – Contribution and quality yield by production lines  
- **Quality Metrics** – First grade %, downgrade %, and overall yield  
- **Reprocessing Analysis** – Repacking ratio and production stability  
- **Product Insights** – Category-wise and denier-level distribution  

---

## 📊 Key Insights

- Observed **quality drift (-14%)**, indicating process inconsistency  
- First-grade production contributes approximately **78%** of total output  
- One unit contributes a higher share (~**56%**), highlighting imbalance  
- Repacking ratio (~**38%**) indicates potential process inefficiencies  
- Variations in line performance suggest scope for optimization
